# remilya.github.io

Static portfolio site for `www.remilya.dev`, built as a lightweight GitHub Pages repo.

## Structure

- `index.html` - page structure
- `styles.css` - visual system and responsive layout
- `main.js` - project data, card rendering, and reveal animations
- `assets/logos/` - copied project logos used by the showcase

## Local Reference Workflow

External project repos can be cloned into `references/` for research while writing portfolio content.
That folder is ignored by git and is not part of the deployed site.
